# 🐍 Semana 01 — Fundamentos Python

Exercícios práticos de lógica básica em Python, desenvolvidos durante minha jornada de transição para a área de tecnologia.

---

## 📚 Conteúdo estudado

- Variáveis e tipos de dados
- Input e output
- Operadores aritméticos e de comparação
- Condicionais (if, elif, else)
- Loops (for, while)
- F-strings

---

## 📁 Arquivos

| Arquivo | Descrição |
|--------|-----------|
| dia1.py | Nome, idade e cidade com f-string |
| dia2.py | Verificador de nota com média |
| dia3.py | Tabuada com for |
| extra1.py | Par ou ímpar com operador % |
| extra2.py | Verificador de maior de idade |
| extra3.py | Classificador de temperatura |
| extra4.py | Login simples com senha |

---

## 🛠️ Como executar

1. Tenha o Python instalado
2. No terminal, navegue até a pasta
3. Execute com: python nome_do_arquivo.py
